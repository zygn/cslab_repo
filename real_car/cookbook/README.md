# Real Car Cookbook
## README


이 Cookbook은 경상대학교 정보통신공학과 CS Lab이 F1TENTH 대회에 참여하면서

생긴 시행착오나, 문제 해결방안을 정리하면서

CS Lab의 내부 세미나 및 정보 참고용으로 제작한 문서 입니다. 

이는 F1TENTH의 가이드라인을 따르고 있으나, **F1TENTH에서 제공하는 표준 빌드 방법은 아님을 밝힙니다.**


대부분의 문서는 f1tenth.org 의 [Build](https://f1tenth.org/build.html) 부분을 참고하여 만들어졌습니다. 

정확한 정보는 f1tenth.org 에서 확인 하시길 바랍니다.

## P.S.

### 1호차 한대에 들어간 부품

 - Traxxas Chassis
 - Nvidia TX2 
 - Orbitty Carrier for NVIDIA® Jetson™ TX2/TX2i
 - Sparkfun 9Dof IMU
 - FESEC 4.12 VESC
 - ZED Stereo Camera
 - Traxxas 11.1v 3S Li-Po Battery
 - F1TENTH Powerboard with DC-IN (12V 5A, 5.5mm * 2.5mm)
 - LCD Digit Voltmeter
 - 7 Port USB 2.0 Hub
 - 부품 고정용 가이드 

### 그외 필요한 도구

 - 드라이버 세트 (샤오미 미지아 멀티 드라이버 세트 추천)
 - Ubuntu 18.04 이상 설치된 노트북
 - Li-Po 배터리 충전기
 - 디버깅용 모니터 (HDMI)
 - 무선 키보드 마우스 세트
 - 양면 테이프
 - 모터 구리스
 - WD-40
